# Cornish Independent NonAligned Group (CING)

The official website for the Cornish Independent NonAligned Group — independent councillors serving Cornwall's communities on Cornwall Council, free from party politics.

**Live site:** [www.cingparty.uk](https://www.cingparty.uk)

## About CING

CING is a group of independent councillors on Cornwall Council. We are not a political party. Our councillors vote on the merits of each issue without a party whip, putting Cornwall's communities first.

- **Rowland O'Connor** — Group Leader, St Columb Major, St Mawgan & St Wenn
- **Anna Thomason-Kenyon** — Deputy Leader, Constantine, Mabe & Mawnan
- **Karen Knight** — Camborne West & Treswithian

## Development

Built with [Hugo](https://gohugo.io/) and [Tailwind CSS](https://tailwindcss.com/). No Node dependencies.

```bash
# Prerequisites: Hugo extended v0.159.1+
hugo server

# Production build
hugo --gc --minify
```

## Deployment

Deployed automatically to GitHub Pages via GitHub Actions on push to `main`. Custom domain: `www.cingparty.uk`.
